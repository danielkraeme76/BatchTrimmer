Program Description:

BatchTrimmer is a desktop application for Windows that allows users to process multiple audio files in batch mode. 

Its main functions include:
- Trimming silence at the beginning and end of audio files based on a configurable threshold.
- Normalizing audio levels using a peak-based method.
- Applying fade-in and fade-out effects with adjustable duration.
- Exporting processed files in various formats (WAV, MP3, FLAC, OGG, AAC) with selectable bit depth and sample rate.
- Providing options for output folder, filename suffix, overwrite handling, and displaying processing status.
